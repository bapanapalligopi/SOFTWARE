# What is Claude Code?

Claude Code is an agentic coding tool that understands your codebase, edits your files, runs commands, and integrates with your existing developer tools to help you get things done faster.

### Available Platforms

* Terminal
* Visual Studio Code
* Claude Desktop App
* Web
* JetBrains IDEs

---

# What Separates Claude Code from Claude?

If you've used Claude.ai before, you might be wondering what makes Claude Code different.

Unlike Claude.ai, Claude Code has direct access to:

* Your files
* Your terminal
* Your entire codebase

Instead of copying and pasting code back and forth, it goes in and does the work itself.

### Key Difference

Claude Code works as an **AI Agent**.

---

# What is an Agent?

An AI Agent is software that can interact with its environment and perform actions to complete a defined goal.

At its core:

* A large language model operates in a loop in real time.
* The agent can use tools and services.
* It can interact with external systems.
* It can even work with other AI agents when needed.

The purpose is to autonomously take actions that help achieve a specific objective.

# Using Claude Code Effectively

To get the most out of Claude Code, keep these three concepts in mind.

## 1. The Context Window

Think of this as Claude's working memory.

* It can hold a large amount of information.
* It cannot hold an entire codebase at once.
* Claude strategically searches your project to find the information it needs.

This is a key part of what makes it agentic.

## 2. It Asks for Permission

By default, Claude Code asks before:

* Running commands
* Editing files
* Making significant changes

You remain in control at every step.

## 3. It Can Make Mistakes

Like any tool, Claude Code isn't perfect.

Possible issues include:

* Misunderstanding your intent
* Introducing bugs
* Creating overly complex solutions

Staying involved helps catch issues early.
