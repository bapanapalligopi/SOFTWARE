# Installing Claude Code

Claude Code is simple to install whether you want to use it in your terminal, on the web, or directly inside your IDE.

---

# Terminal Installation

Claude Code can be installed on multiple operating systems.

### macOS, Linux, and WSL

Use the installation command provided by Anthropic:

* Install Claude Code using the `curl` command.
* Homebrew is also supported via `brew install`.
* Homebrew installations do **not** support automatic updates.

### Windows

You have several installation options:

* PowerShell using `Invoke-RestMethod`
* Command Prompt using `curl`
* `winget` package manager

> Note: Like Homebrew, the `winget` installation method does not support automatic updates.

### Launching Claude Code

After installation:

1. Open a terminal.
2. Navigate to your project folder.
3. Run:

```bash
claude
```

If the command isn't recognized:

* Restart your terminal.
* Verify the installation completed successfully.

### First-Time Setup

During setup, Claude Code will ask you to:

* Choose a color theme
* Sign in to your Claude account
* Select an authentication method

Available login methods:

* Claude Pro
* Claude Max
* Claude Enterprise
* API Key

If your organization uses Claude Enterprise, select that option during login.

Alternatively:

* Click the Claude logo in the sidebar if visible.

# Which One Should You Use?

The best option depends on your workflow.

## Terminal

Best for:

* Power users
* Early access to new features
* Maximum flexibility

### Pros

* Latest features arrive first
* Full control
* Lightweight experience

---

## VS Code & JetBrains

Best for:

* Developers who spend most of their time in an IDE
* Integrated coding workflows

### Pros

* Familiar environment
* Tight editor integration
* Minimal context switching

---

## Desktop

Best for:

* Multitasking
* Background execution
* Managing multiple projects

### Pros

* Dedicated workspace
* Easy project management
* Cloud environment support

---

## Web

Best for:

* Remote development
* Working from any device
* GitHub-based projects

### Pros

* No local installation required
* Accessible anywhere
* Repository-based workflow

---

# Recap

### Ways to Use Claude Code

You can use Claude Code through:

* Terminal
* Visual Studio Code
* JetBrains IDEs
* Claude Desktop
* Web Browser

### Choosing the Right Platform

* Use **Terminal** for the newest features.
* Use **VS Code or JetBrains** for a fully integrated coding experience.
* Use **Desktop** for background workflows and project management.
* Use **Web** for remote access and GitHub-based development.

No matter which option you choose, Claude Code provides the same core capabilities: understanding your codebase, taking action, and helping you build software more efficiently.
